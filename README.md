# ネウチ！ — 価値観を読む逆オークション

ブラウザで2〜30人が同時に遊べるオンラインパーティゲームです。
**npm install 不要・依存ライブラリゼロ**。Node.js だけで動きます。

## 遊び方(30秒ルール)

1. 商品が出ます。こっそり金額を書いてください。
2. **一番安い人だけ**が、その額で買えます。ただし**同じ額の人がいたら全員無効**。
3. 商品の価値は、あなたの秘密の**「価値観カード」**で変わります。
4. 10ラウンド後、いちばんお金持ちが勝ち。

## 起動方法

Node.js v18 以上が入っているPCで:

```
node server.js
```

と実行し、ブラウザで http://localhost:3000 を開くだけです。

- **他の人の参加**: 同じネットワーク(Wi-Fi/LAN)にいる人は
  `http://<起動したPCのIPアドレス>:3000` を開き、画面の「合言葉」(4文字)で入室します。
  IPアドレスは Windows なら `ipconfig`、Mac/Linux なら `ifconfig` で確認できます。
- **ポート変更**: `PORT=8080 node server.js`
  (Windowsのコマンドプロンプトは `set PORT=8080 && node server.js`)
- **人数が足りないとき**: ロビーでホストが「🤖 AIプレイヤー追加」を押すとAIが参加します。
  1人でもAI相手に練習できます。

## ファイル構成

```
server.js          ... ゲームサーバー本体 (これ1つで完結)
public/index.html  ... クライアント (自動配信されます)
test.js            ... 自動テスト (node test.js)
test30.js          ... 30人負荷テスト (node test30.js)
```

## ゲームの仕組み (ネタバレ注意)

- 各プレイヤーに秘密の**価値観カード2枚**が配られ、商品タグごとの価値倍率が変わります。
  他人の入札額から「あいつは動物マニアでは?」と推理するのが本体です。
- 毎ラウンド**号外ニュース**で市場が変動(グルメブーム、バブル、闇オークション等)。
- 同額バッティングで流れた商品は**価値×1.5で次ラウンドに持ち越し**(ジャックポット)。
- 入札には参加費5G。強気の全ツッパも、様子見のパスも戦略です。
- 7人以上では商品が複数レーンに並び、「どこに行くか」の読み合いが加わります。
- 最終ラウンド後、**他人の価値観を当てる**と +100G。
- 最終資産 = 現金 + 商品(基準値 × 価値観倍率 × 同タグ収集ボーナス)。

## 補足

- 通信が切れても、同じ端末・同じブラウザなら「前回のゲームに再接続する」で復帰できます。
- ゲーム中の途中参加はできません(観戦したい人は次のゲームから)。
- サーバーはメモリ上で動作し、1時間動きのないルームは自動削除されます。
